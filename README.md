## The Produce Shop
This page highlights my passion for produce while practicing key HTML and CSS concepts.
HTML:
- `<nav>` for site navigation
- `<header>` for introductory content
- `<section>` blocks for produce images and videos
- `<article>` to highlight featured produce with expandable `<details>`
- `<blockquote>` for thematic or inspirational quotes
CSS:
- Sticky navbar with hover effects and full-width background
- Responsive image grid using floats and clearfix
- Font pairing via Google Fonts (Roboto + Mukta)
- Animated header text using `@keyframes`
- Styled contact form with focus transitions and hover feedback
- Section-specific background colors tied to produce categories
- Clean footer layout using table-based alignment

## Movie Theater Site
This page is an example home page for a movie theater that is styled using many Bootstrap components including a navbar, form, cards, buttons and dropdowns.
- How to build a full page layout using Bootstrap’s grid system
- How to combine Bootstrap components with custom CSS
- How to create responsive, mobile‑friendly sections without writing complex CSS
- How to structure content for readability and maintainability
